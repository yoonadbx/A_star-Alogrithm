# 1. 第一次完整的用class 来创建变量环境，体会到class的几个优点。
# 2. 体会到枚举类型的 Eunm的优点：
# 1）枚举类型可以看作是一种标签或是一系列常量的集合。
# 2）使用类获取属性的时候，输出的不是它的数值，而是会显示枚举项，这样更能增加代码的可读性。
# 3）不允许在类外直接修改枚举项的值。
# 4）不存在 key 相同的枚举项（类变量）。
# 5）枚举可以相同的值，但是值相同的各项 Key 都会当做别名。
